## 🎯 项目简介

AutoSignIn 是一套个人自动化基础设施，涵盖网络代理管理、Android 自动化任务执行、远程通话/短信管理等功能。整个系统运行在家庭网络环境中，由一台 OpenWrt 旁路由和一台 Android 设备协同工作。

**核心能力：**
- 🌐 透明代理 — Xray TProxy 实现翻墙 + 公司内网分流 + 国内直连
- 🤖 Android 自动化 — AutoJS6 执行签到、发帖、打卡等定时任务
- 📞 通话/短信管理 — Termux 上报来电和短信到 Bark 推送
- 🛡️ Web 管理面板 — 浏览器管理 Xray 服务、配置、路由规则

## 📦 模块说明

### 1. xray-panel — Xray 管理面板

| 项目 | 说明 |
|------|------|
| **位置** | `xray-panel/` |
| **部署目标** | 旁路由 192.168.50.2 |
| **运行环境** | uhttpd + CGI (Shell) |
| **访问地址** | `http://192.168.50.2/xray/` |

**功能：**
- 服务状态监控（PID、内存、连接数、负载）
- 连通性测试（Google/内网域名/内网IP/百度）
- 配置文件在线编辑与语法验证
- 路由规则快速编辑（office/direct 域名和 IP 段）
- 日志查看与实时刷新（支持按类型过滤）
- GeoIP/GeoSite 规则库更新（手动/定时）
- 面板密码管理
- **自动初始化**：登录/刷新后自动加载测试结果、配置、路由规则

**文件结构：**
```
xray-panel/
├── www/
│   ├── xray-panel/index.html    # 前端面板（HTML + 纯 JS）
│   └── cgi-bin/xray-api         # 后端 API（Shell CGI）
└── update-geodata.sh            # GeoIP/GeoSite 更新脚本
```

**部署命令：**
```bash
sshpass -p 'PASSWORD' scp xray-panel/www/xray-panel/index.html root@192.168.50.2:/www/xray/index.html
sshpass -p 'PASSWORD' scp xray-panel/www/cgi-bin/xray-api root@192.168.50.2:/www/cgi-bin/xray-api
```

## 🚀 部署指南

### Xray 管理面板部署

```bash
# 1. 部署前端
sshpass -p 'PASSWORD' scp xray-panel/www/xray-panel/index.html root@192.168.50.2:/www/xray/index.html

# 2. 部署后端 API
sshpass -p 'PASSWORD' scp xray-panel/www/cgi-bin/xray-api root@192.168.50.2:/www/cgi-bin/xray-api

# 3. 部署 GeoData 更新脚本
sshpass -p 'PASSWORD' scp xray-panel/update-geodata.sh root@192.168.50.2:/usr/local/xray/update-geodata.sh
```

### Android 服务部署

参见各模块独立文档：
- `doc/Android Telecom API 服务文档.md`
- `doc/AutoJS API 服务文档.md`
- `doc/Android 初始化.md`

### Xray 配置修改流程

1. 打开面板 `http://192.168.50.2/xray/`
2. 点击「加载配置」或等待自动加载
3. 在配置编辑器中修改
4. 点击「验证语法」确认无误
5. 点击「保存配置」
6. 点击「重启」使配置生效

## 🔧 日常运维

### 常用操作

| 操作 | 方式 |
|------|------|
| 查看 Xray 状态 | 面板首页或 `ssh root@192.168.50.2 '/etc/init.d/xray status'` |
| 重启 Xray | 面板「重启」按钮或 `ssh ... '/etc/init.d/xray restart'` |
| 更新 GeoIP/GeoSite | 面板「规则库更新」或 `ssh ... '/usr/local/xray/update-geodata.sh'` |
| 查看代理日志 | 面板「日志」区域，支持按类型过滤 |
| 测试连通性 | 面板自动测试或手动点击「运行测试」 |
| 远程执行签到脚本 | `curl -X POST http://ANDROID_IP:8080/run -H "Authorization: Bearer TOKEN" -d '{"script":"audi_signin.js"}'` |

### 故障排查

| 问题 | 排查方式 |
|------|---------|
| 面板频繁跳转登录 | 检查是否有并发请求竞态，后端仅支持单 token 文件 |
| 翻墙不通 | 面板测试 Google → 查日志 → 检查 VPS 状态 |
| 公司内网不通 | 面板测试内网域名/IP → 检查 office 出口配置 |
| Android 任务未执行 | 检查 AutoJS API 服务状态 → 看门狗日志 |
| WireGuard 断开 | 检查 wireguard-watchdog.sh 是否在运行 |

### 注意事项

- **后端 CGI 并发限制**：Xray 面板后端是 Shell CGI 脚本 + 单文件 token，不能并行发送多个 API 请求，否则会出现 token 验证竞态
- **GeoData 更新**：建议设置每周自动更新一次（面板中可配置 cron）
- **Android 开机顺序**：Magisk → AutoJS API → WireGuard → Termux Boot → Telecom API
- **密码安全**：面板密码存储在 `/usr/local/xray/.panel_pwd`，首次使用请修改默认密码 `admin`
